# bt-ui
vue components build with vue
## api
[演示文档](http://frontman.cn/bt-ui)

## install
```shell
npm install bt-ui -S
```
## quickstart
``` javascript

import Vue from 'vue'
import btui from 'bt-ui'

Vue.use(btui)

// or
import {
  Select,
  Button
  // ...
} from 'bt-ui'

Vue.component(Select.name, Select)
Vue.component(Button.name, Button)

```
## Browser Support
chrome、firefox、暂不支持IE。
